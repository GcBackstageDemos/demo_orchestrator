# demo_orchestrator

Example on how to integrate Orchestrator with Backstage Catalog

## How to configure 

. Update *tssc-developer-hub-app-config* to enable external API calls:
     
      backend:
        auth:     
          externalAccess:
            - type: static
              options:
                  token: pleasech4angem3
                  subject: external
      

. Import this xref:./templates/location.yml [location.yml] to Developer Hub using the console

. Update xref:./workflow/manifests/00-secret_software-template-workflow-secrets.yaml[00-secret_software-template-workflow-secrets.yaml] with your Developer Hub URL

. Apply the manifests in the manifest folder
